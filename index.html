MSPFA to AO3

What is this?

This is a pretty silly converter to put MSPaint Fan Adventure Adventures into Archive of our own. Credit to EtchJetty for coming up with the adventure in a story concept in the first place. There are a few limitations; since HTML in AO3 is a bit of a pain, currently only the colors in the offical AO3 Homestuck Workskin are supported (as well as #00ff00). Also any custom CSS on the adventure itself is out the window as well, sorry. It is posssible to convert the adventure's css but it has to be done manually, since AO3 is also a bit a pain regarding CSS. Also no text backgrounds, fonts, text size or MSPFA user tags, due to the CSS being a pain, and no flashes, mp4 embeds or youtube videos, simply because AO3 removes them.
